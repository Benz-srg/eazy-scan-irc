# ไฟล์ตัวอย่างสำหรับทดสอบ

| ไฟล์ | ใช้ทำอะไร |
|---|---|
| `sample-requirement-th.m4a` | เสียง requirement ภาษาไทย (~15 วิ) — ลากเข้าหน้า Workspace เพื่อทดสอบ flow เต็ม (อัปโหลด → ถอดเสียง → วิเคราะห์) |
| `sample-transcript-th.txt` | ข้อความ transcript เดียวกัน — ใช้ทดสอบเฉพาะส่วน LLM ผ่าน `POST /api/reanalyze` โดยไม่ต้องถอดเสียง (เร็วสุดสำหรับตรวจ AI) |

เนื้อหา: ลูกค้าร้านค้าปลีก 3 สาขา + ออนไลน์ ต้องการระบบจัดการสินค้า/สต็อก/ออเดอร์รวมศูนย์,
แดชบอร์ดยอดขาย, แจ้งเตือน LINE, รับบัตรเครดิต/พร้อมเพย์ — งบยังไม่ฟิกซ์, อยากใช้ในไตรมาสนี้

ดูวิธีทดสอบเร็วที่ส่วน **"สำหรับผู้ตรวจ (Reviewer Quick Start)"** ใน `README.md` หลัก
