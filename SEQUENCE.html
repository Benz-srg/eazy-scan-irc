มี MongoDB

  • ประวัติเก็บ ถาวร — ข้ามเครื่อง / หลัง refresh ก็ยังอยู่
  • เปิดผลเก่าจาก /results/[id] ได้ทุกเมื่อ
  • row “กำลังประมวลผล” อยู่ฝั่ง server — งานวิ่งต่อแม้ปิดหน้า
  • export DOCX/PDF จากผลเก่าได้

ไม่มี DB (localStorage)

  • ประวัติเก็บใน เบราว์เซอร์เท่านั้น — ล้าง browser / เปลี่ยนเครื่อง = หาย
  • เปิดผลเก่าจาก /results/[id] ไม่ได้ (ไม่มีที่เก็บผล)
  • row “กำลังประมวลผล” ยังเห็น (optimistic ฝั่ง client)
  • วิเคราะห์ + ดูผลในรอบปัจจุบันได้ปกติ

เมื่อเกิดข้อผิดพลาด

ไม่แสดงข้อมูลตัวอย่างปลอม — แจ้ง error ที่อ่านเข้าใจ + บอกวิธีแก้
1
เรียกถอดเสียง / วิเคราะห์
2
ล้มเหลว — เช่น key ผิด (401), Claude CLI ไม่ล็อกอิน, Whisper :8000 ดับ
3
จัดประเภท error → อัปเดต status=error + ข้อความไทยที่ชัด
4
หน้า error: “ถอดเสียงไม่สำเร็จ” / “เชื่อมต่อ AI วิเคราะห์ไม่ได้” + วิธีแก้
ไม่มี DB ก็ยังเห็น row ฝั่ง client เปลี่ยนเป็น “ผิดพลาด” เอง
